# MOUVE Operational Dashboard

Daily KPI metrics for Louise and Georgia.

**Stack:** Static HTML/CSS/JS + Chart.js, hosted on Vercel with password protection.

**Data:** `operations-data.json` updated daily at 06:35 by n8n workflow `OPS-DASHBOARD-V1`.

**KPIs:**
- New Leads Won (7d / 30d) — GHL Pipeline 1
- Children Enrolled — Airtable (from DSP sync)
- Total Classes Running — Airtable (from DSP sync)
- LF Active Members / Trials — Airtable
- LF Trial Conversion Rate — calculated
